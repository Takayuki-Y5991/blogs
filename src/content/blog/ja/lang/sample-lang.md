# Clojure 入門

Clojureは関数型プログラミング言語です。

## 特徴

- JVM上で動作
- Lispの方言
- 不変データ構造

## サンプルコード

```clojure
(defn hello [name]
  (str "Hello, " name "!"))
```

## まとめ

Clojureは強力な言語です。
