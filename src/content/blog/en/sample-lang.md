# Introduction to Clojure

Clojure is a functional programming language.

## Features

- Runs on the JVM
- A dialect of Lisp
- Immutable data structures

## Sample Code

```clojure
(defn hello [name]
  (str "Hello, " name "!"))
```

## Conclusion

Clojure is a powerful language.
